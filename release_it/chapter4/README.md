## The 9 Stability Patterns

| # | Pattern | Primary Benefit | Best For |
|---|---------|-----------------|----------|
| 1 | **Circuit Breaker** | Prevents cascade failures | External APIs, databases |
| 2 | **Bulkhead** | Limits blast radius | Critical resources |
| 3 | **Timeout** | Prevents resource accumulation | All external calls |
| 4 | **Handshake** | Backpressure | High-volume systems |
| 5 | **Decoupling (Middleware)** | Decoupling | Distributed systems |
| 6 | **Fail Fast** | Early detection | Input validation |
| 7 | **Let It Crash** | Clean recovery | Stateless services |
| 8 | **Bulkheads (Redux)** | Multi-level isolation | Architecture design |
| 9 | **Stable Topology** | Reduced dependencies | System architecture |
